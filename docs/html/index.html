fintulib.cloud¶

Some helpers for accessing cloud objects are in this submodule.

fintulib.cloud.gcs¶

This module contains helpers to upload / download files and python objects to Google Cloud Storage.

Python objects are serialized with pickle.

Note that Python objects need to fit in memory if uploaded directly. Otherwise, a temporary local file is required.
